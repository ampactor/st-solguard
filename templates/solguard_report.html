Methodology

SolGuard's autonomous pipeline: narrative-driven target selection ensures we scan what matters, not random repos.

1. Signal Collection

Parallel collection from GitHub API (new Solana repos, star velocity), Solana RPC (TPS, program activity), blog scraping (Helius, Jito, Marinade), and DeFiLlama (TVL, protocol rankings).

→
↓

2. Narrative Synthesis

LLM identifies growth narratives from aggregated signals, assigning confidence scores and linking to active repositories.

→
↓

3. Security Scanning

Static analysis: 10 regex + 3 AST vulnerability patterns. Deep review: multi-turn LLM agent investigates trust boundaries, fund flows, and state transitions.

→
↓

4. Adversarial Validation

A separate LLM pass attempts to disprove each finding — reading cited code, checking mitigations, verifying attack reachability.

→
↓

5. Cross-Reference

Risk scoring: severity_weight * validation_multiplier * narrative_confidence. Findings linked to narratives for ecosystem context.
